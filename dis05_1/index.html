  • How many bits are used to address each chip in a memory that is 64K bytes large and made up of 32K by 1 byte memory chips? How many bits are used as the chip select in this memory?

Detour Sign

Instead of wrapping around from fully-lit to the all off state, the sign should reverse lighting direction until all lights are off. (see explanation on board)
